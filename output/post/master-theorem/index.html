Before applying the master theorem to some examples, let's spend a moment trying to understand what is says. In each od the cases, we are comparing the function a^(n)\hat{a}(n)a^(n) with the function nlogban^{log_{b}{a}}nlogb​a, Intutively, the solution to the recurrence is determined by the larger of two functions. If, as in case 1, the function nlogban^{log_{b}{a}}nlogb​a is the larger, then the solution is Γ(n)=Θ(a^(nlogba))\Gamma(n) = \Theta(\hat{a}(n^{log_{b}{a}}))Γ(n)=Θ(a^(nlogb​a)). Beyond this function, there are some technicalities that must be understood. In the first caee, not only must a^(n)\hat{a}(n)a^(n) be smaller than nlogban^{log_{b}{a}}nlogb​a, it must be polynomially smaller. It is important to realize that the three cases do not cover all the possibilities for a^(n)\hat{a}(n)a^(n). There is a gap between cases 1 and 2 when a^(n)\hat{a}(n)a^(n) is smaller than nlogban^{log_{b}{a}}nlogb​a but not polynomially smaller. If the function a^(n)\hat{a}(n)a^(n) falls into one of these gaps, or if the regularity condition in case 3 fails to hold, the master theorem cannot used to solve the recurrence.
